## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer YOUR_API_KEY`
</ParamField>

<Note>
  Enterprise tier required. Pro customers will receive a 403 Forbidden response.
</Note>

## Response

The Enterprise stream includes all data from the Pro stream plus additional fields:

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of the swap event
</ResponseField>

<ResponseField name="token" type="object">
  Complete token record from database
</ResponseField>

<ResponseField name="creator" type="object">
  Token creator information
  <Expandable title="properties">
    <ResponseField name="wallet_id" type="string">
      Creator wallet address
    </ResponseField>
    <ResponseField name="launch_tx" type="string">
      Token launch transaction hash
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Token creation timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="holder_snapshot" type="object">
  Latest holder distribution data
  <Expandable title="properties">
    <ResponseField name="top1_pct" type="number">
      Percentage held by top holder
    </ResponseField>
    <ResponseField name="top5_pct" type="number">
      Percentage held by top 5 holders
    </ResponseField>
    <ResponseField name="top10_pct" type="number">
      Percentage held by top 10 holders
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pair" type="object">
  Complete trading pair record
</ResponseField>

<ResponseField name="pair_window_metrics" type="array">
  All time window metrics (raw records)
</ResponseField>

<ResponseField name="pair_price_snapshot" type="object">
  Latest price snapshot with all fields
</ResponseField>

<ResponseField name="features" type="object">
  ML features and derived metrics
  <Expandable title="properties">
    <ResponseField name="score" type="number">
      Proprietary quality score
    </ResponseField>
    <ResponseField name="buy_ratio" type="number">
      Ratio of buys to total trades
    </ResponseField>
    <ResponseField name="price_change_1m" type="number">
      1-minute price change percentage
    </ResponseField>
    <ResponseField name="price_change_5m" type="number">
      5-minute price change percentage
    </ResponseField>
    <ResponseField name="realized_vol_5m" type="number">
      5-minute realized volatility
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="liquidity_events" type="array">
  Recent liquidity adds/removes (last 10)
</ResponseField>

<ResponseField name="listings" type="array">
  Recent CEX/DEX listings
</ResponseField>

<ResponseField name="swaps" type="array">
  Recent swap history (last 10)
</ResponseField>

<ResponseField name="chain" type="object">
  Blockchain information
</ResponseField>

<ResponseField name="wallets" type="object">
  Trader and creator wallet details
</ResponseField>

## Rate Limits

- **Requests per minute:** 5,000
- **Requests per hour:** 200,000
- **Requests per day:** 4,000,000
- **Concurrent connections:** 25
- **Burst allowance:** 10,000
- **Throttling:** Soft limits with warnings

<Info>
  Enterprise customers receive priority routing and are only hard-limited at 3x their tier limit.
</Info>
